?? FLUJO COMPLETO PARA NUEVOS DESTINOS:

1. EXPLORAR DESTINO:
   node scripts/activity-manager.js destination-info rome
   node scripts/activity-manager.js list-templates rome

2. CREAR VIAJE:
   node scripts/trip-planner.js create "Roma Imperial" rome 2025-10-15 2025-10-18 800

3. APLICAR TEMPLATE:
   node scripts/trip-planner.js list (obtener trip ID)
   node scripts/activity-manager.js apply-template rome_ancient TRIP_ID

4. REVISAR Y EXPORTAR:
   node scripts/trip-planner.js show TRIP_ID
   node scripts/trip-planner.js export TRIP_ID txt
